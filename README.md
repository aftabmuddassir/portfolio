A simple static portfolio built with HTML/CSS and Font Awesome.

## Local preview

- Open `index.html` directly in your browser, or
- Use a lightweight server:

```bash
python3 -m http.server 8080
```
Then visit `http://localhost:8080`.

## Customize

- Update content in `index.html` (About, Services, Projects, Contact)
- Replace images in `images/`
- Edit styles in `style.css`

## Contact form

The contact form posts to a Google Apps Script endpoint configured in `index.html`. Replace `scriptURL` with your own Apps Script URL if needed.

## Deploy

- GitHub Pages: push to `main` and enable Pages on the repository
- Netlify/Vercel: drag-and-drop the folder or point to `index.html`
